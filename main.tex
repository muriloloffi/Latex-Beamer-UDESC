\begin{frame}
    \frametitle{Alguns trabalhos de Edsger Dijkstra}
    \begin{itemize}
        \item Algoritmo de caminho mínimo em grafos e redescobriu o algoritmo Prim.
        \item Sistema Operacional THE, tipo BATCH, com camadas de abstração e suporte a multitasking.
        \item Variável de semáforo.
        \item Trabalhou no primeiro compilador da linguagem ALGOL 60.
        \item Introduziu o conceito de pilha em programas recursivos.
        \item Pai da programação estruturada - Uso de subrotinas, laços de repetição etc.
        \item Defensor do uso de metodologias matemáticas na programação - Verificação formal.
        \item Identificou e apresentou a solução de um problema de exclusão m\'utua em um sistema concorrente.
        \item Resumindo... fez de tudo.
    \end{itemize}
\end{frame}
